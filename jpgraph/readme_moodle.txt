Instructions for downloading and build jpgraph for BLOCKS_OBU_ANALYTICS

1. Download the latest QPL 1.0 version into temporary folder from https://jpgraph.net/download/ 
2. Extract folders to a temporary location
3. Delete the src folder from this location (ensure it is backed up in sorce control or a zip file)
4. Move the src folder and VERSION file into this folder
5. Update thirdpartylibs.xml with the latest version.
6. Test
